# Network Topology Design

A simple network simulation built using **Cisco Packet Tracer**. The project connects two LANs using a **Cisco 2811 router**, configured with basic IP addressing and routing to enable full communication between all devices.

* **Key Features:** Dual-LAN connectivity, Router Configuration, IP Addressing.
